I built IDA (https://www.project-ida.com) — a web app that generates interior concepts from a room photo + user preference profile.

The platform collects declarative preferences (questionnaires, semantic differentials) and behavioral signals (swipe choices, response times) and maps them — together with optional Big Five (IPIP-NEO-120) scores — to generation parameters for a diffusion-based image model (Vertex AI).

Tech: Next.js, GCP Cloud Run, Cloud SQL. Research data with informed consent; pseudonymized participant IDs.

- Not construction documentation
